1. Description of each classes

Simulator:
    A simulator that simulate a blackjack game

Deck:
    A class that hold a deck of blackjack game

Card:
    A card class represent general poker card

Player:
    Player class that play blackjack game in the simulator

Dealer:
    Dealer class that play blackjack game in the simulator

2. How to compile and run the code
$ make build
$ make run


3. How to play
Simply follow the instruction from the program
